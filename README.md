# gexhook
gexhook is a small trainer for Gex 1.

![A showcase of the free camera mode and debug info](img/freecam_debug.png)

## Installation
Before starting, make sure you have the **GOG version of Gex 1**. This will not work with the Gex Trilogy or any console version of the game.

Download and place [ddraw.dll from Ultimate-ASI-Loader](https://github.com/ThirteenAG/Ultimate-ASI-Loader/releases/download/Win32-latest/ddraw-Win32.zip) into your game directory.
Next, download the latest release of gexhook, and place `gexhook.asi` in your game directory.

## Features
 - Debug menu (open with 'O')
 - Free camera mode
 - Level select
